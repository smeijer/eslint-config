# eslint-config

Shared [ESLint config](eslint) for my projects

## Install

```sh
npm install --save-dev @smeijer/eslint-config
npx @smeijer/eslint-config
```

## Usage

`eslintrc.json`

```json
{
  "extends": "@smeijer/eslint-config"
}
```

[eslint]: https://eslint.org/docs/latest/extend/shareable-configs
